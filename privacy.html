개인정보처리방침

시행일: 2026년 1월 1일  |  최종 수정일: 2026년 5월 12일

오늘의사주(이하 "회사" 또는 "서비스")는 사용자의 개인정보를 매우 중요하게 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 회사가 수집하는 정보의 종류, 이용 목적, 보관 기간 및 사용자의 권리에 대해 설명합니다.

💙 2026년 5월 업데이트: 사주 기반 소셜 매칭 기능 추가에 따라 개인정보처리방침이 업데이트되었습니다.

  • 서비스 개선 및 오류 수정
  • 이용권 구매 및 결제 처리
  • 신고·차단 등 안전한 서비스 환경 유지

4. 개인정보 제3자 제공

회사는 사용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만, 서비스 제공을 위해 아래 업체와 협력합니다.

제공 업체 제공 목적 제공 항목
Anthropic (Claude AI) AI 사주 분석 생년월일, 성별, 분석 요청 내용 (익명화)
Cloudflare API 프록시 서버 운영 암호화된 요청 데이터
RevenueCat 인앱 결제 처리 기기 식별자 (익명)
Supabase NEW 매칭 서비스 데이터 저장 및 실시간 채팅 매칭 프로필, 위치(시/도), 채팅 메시지, 프로필 사진

5. 매칭 서비스 관련 안내 NEW

  • 매칭 프로필에 입력한 정보(닉네임, 나이, 프로필 사진, 오행 타입, 사주 캐릭터 등)는 다른 사용자에게 공개됩니다.
  • 위치 정보는 시/도 단위로만 표시되며, 정확한 주소는 수집·표시하지 않습니다.
  • 채팅 메시지는 암호화되어 저장되며, 매칭된 두 사용자만 열람 가능합니다.
  • 부적절한 사용자를 신고 또는 차단할 수 있습니다.
  • 매칭 프로필은 언제든지 앱 내 설정에서 삭제할 수 있습니다.
  • 미성년자(만 18세 미만)의 매칭 서비스 이용을 금지합니다.

6. AI 서비스 이용 안내

본 서비스는 Anthropic의 Claude AI를 활용하여 사주 분석 결과를 생성합니다. AI에게 전달되는 정보는 생년월일, 성별 등 최소한의 정보만 포함되며, 이름 등 민감한 개인정보는 포함되지 않습니다.

AI가 생성한 결과는 오락 및 참고 목적으로만 제공되며, 의료, 법률, 재정적 조언을 대체하지 않습니다.

7. 사용자의 권리

  • 열람권: 저장된 개인정보 확인 요청 가능
  • 삭제권: 앱 내 "MY → 내 정보 삭제" 또는 앱 삭제 시 기기 내 데이터 삭제. 매칭 프로필은 앱 내 매칭 설정에서 삭제 가능
  • 이의 제기권: 개인정보 처리에 대해 이의 제기 가능
  • 처리 제한권: 특정 목적의 데이터 처리 제한 요청 가능
  • 매칭 프로필 비공개: 매칭 서비스 내 프로필 활성화/비활성화 선택 가능

권리 행사는 아래 연락처로 요청하시면 7일 이내 처리됩니다.

8. 개인정보 보호 조치

  • 모든 데이터 전송은 HTTPS/TLS 암호화 적용
  • 기기 내 데이터는 iOS Keychain 또는 AsyncStorage를 통해 보호
  • 관상 분석용 얼굴 사진은 서버에 저장하지 않으며 분석 후 즉시 파기
  • 매칭 프로필 사진 및 채팅 메시지는 Supabase 암호화 서버에 저장
  • 결제 정보는 직접 저장하지 않으며 결제 대행사(RevenueCat)를 통해 처리
  • 매칭 서비스의 Row Level Security(RLS) 정책으로 본인 데이터만 접근 가능

9. 아동 및 미성년자 보호

본 서비스의 기본 기능은 만 14세 미만 아동을 대상으로 하지 않습니다. 매칭 서비스는 만 18세 이상만 이용 가능합니다. 미성년자의 개인정보가 수집된 것을 인지한 경우 즉시 삭제 조치합니다.
