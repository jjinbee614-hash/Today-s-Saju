AI 사주명리학 서비스

개인정보처리방침

시행일: 2026년 1월 1일  |  최종 수정일: 2026년 4월 29일

오늘의사주(이하 "회사" 또는 "서비스")는 사용자의 개인정보를 매우 중요하게 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 회사가 수집하는 정보의 종류, 이용 목적, 보관 기간 및 사용자의 권리에 대해 설명합니다.

1. 수집하는 개인정보 항목

수집 항목 수집 방법 필수 여부
생년월일, 출생 시간 사용자 직접 입력 필수 (사주 분석용)
성별 사용자 직접 입력 필수 (사주 분석용)
이름 (선택) 사용자 직접 입력 선택 (작명·이름감정용)
얼굴 사진 카메라 또는 갤러리 선택 (관상 분석용)
기기 식별자 (익명) 자동 수집 서비스 운영용

📸 얼굴 사진 관련 중요 안내:

관상 분석을 위해 업로드된 얼굴 사진은 분석 즉시 삭제되며, 어떠한 형태로도 저장·보관·공유되지 않습니다. 사진은 AI 분석을 위해 일회성으로만 사용됩니다.

2. 개인정보 이용 목적

  • 사주명리학 기반 운세, 사주풀이, 궁합, 택일, 작명 서비스 제공
  • 관상 분석 서비스 제공 (사진은 분석 후 즉시 삭제)
  • 사용자 맞춤 결과 생성 및 표시
  • 서비스 개선 및 오류 수정
  • 이용권 구매 및 결제 처리

3. 개인정보 보관 기간

항목 보관 기간
생년월일, 출생 시간, 성별 기기 내 저장 (사용자가 직접 삭제 가능)
얼굴 사진 분석 완료 즉시 삭제 (서버 미저장)
운세 분석 결과 캐시 당일 자정까지 (기기 내 임시 저장)
결제 정보 관련 법령에 따라 5년 보관

4. 개인정보 제3자 제공

회사는 사용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만, 서비스 제공을 위해 아래 업체와 협력합니다.

제공 업체 제공 목적 제공 항목
Anthropic (Claude AI) AI 사주 분석 생년월일, 성별, 분석 요청 내용 (익명화)
Cloudflare API 프록시 서버 운영 암호화된 요청 데이터
RevenueCat 인앱 결제 처리 기기 식별자 (익명)

5. AI 서비스 이용 안내

본 서비스는 Anthropic의 Claude AI를 활용하여 사주 분석 결과를 생성합니다. AI에게 전달되는 정보는 생년월일, 성별 등 최소한의 정보만 포함되며, 이름 등 민감한 개인정보는 포함되지 않습니다.

AI가 생성한 결과는 오락 및 참고 목적으로만 제공되며, 의료, 법률, 재정적 조언을 대체하지 않습니다.

6. 사용자의 권리

  • 열람권: 저장된 개인정보 확인 요청 가능
  • 삭제권: 앱 내 "MY → 내 정보 삭제" 또는 앱 삭제 시 기기 내 데이터 삭제
  • 이의 제기권: 개인정보 처리에 대해 이의 제기 가능
  • 처리 제한권: 특정 목적의 데이터 처리 제한 요청 가능

권리 행사는 아래 연락처로 요청하시면 7일 이내 처리됩니다.

7. 개인정보 보호 조치

  • 모든 데이터 전송은 HTTPS/TLS 암호화 적용
  • 기기 내 데이터는 iOS Keychain 또는 AsyncStorage를 통해 보호
  • 얼굴 사진은 서버에 저장하지 않으며 분석 후 즉시 파기
  • 결제 정보는 직접 저장하지 않으며 결제 대행사(RevenueCat)를 통해 처리

8. 아동 개인정보 보호

본 서비스는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동의 개인정보가 수집된 것을 인지한 경우 즉시 삭제 조치합니다.

9. 개인정보처리방침 변경

본 방침은 법령 또는 서비스 변경에 따라 업데이트될 수 있습니다. 변경 시 앱 내 공지 또는 본 페이지를 통해 사전 안내합니다.
